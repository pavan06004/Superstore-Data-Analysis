# Superstore Data Analysis Project

## Project Overview
This project analyzes the Superstore dataset to identify business insights related to sales, profit, customers, products, and regional performance.

The project follows an end-to-end data analytics workflow using SQL, Python, and Power BI.

---

## Tools Used
- SQL (MySQL)
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Excel

### Power BI Dashboard

Created interactive dashboards:

### 1. Executive Dashboard
Includes:
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Sales by Region
- Profit by Region
- Monthly Sales Trend
- Sales by Category
- Interactive slicers

### 2. Product Analysis Dashboard
Includes:
- Sales by Category
- Profit by Category
- Sales by Sub-Category
- Top 10 Products by Sales
- Quantity by Category
- Interactive slicers

### 3. Customer Analysis Dashboard
Includes:
- Top 10 Customers by Sales
- Top 10 Customers by Profit
- Sales by Segment
- Profit by Segment
- Orders by Segment
- Interactive slicers
