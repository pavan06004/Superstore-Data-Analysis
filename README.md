# HR Analytics Analysis

## Project Overview

This project presents an end-to-end HR Analytics solution built using **SQL, Python, and Power BI**. The objective is to analyze employee data, identify workforce trends, and uncover the key factors contributing to employee attrition. The project follows a complete data analytics workflow, from data preparation to interactive dashboard development.

---

## Objectives

* Analyze employee attrition and workforce trends.
* Identify factors affecting employee retention.
* Explore salary, satisfaction, and overtime patterns.
* Build interactive dashboards for HR decision-making.

---

## Tech Stack

* **SQL (MySQL)** – Database creation, data transformation, and business analysis
* **Python** – Data cleaning, exploratory data analysis (EDA), feature engineering, and statistical analysis
* **Power BI** – Interactive dashboards and business insights
* **Git & GitHub** – Version control and project management

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Validation

5. Feature Engineering
6. Statistical Analysis

## Dashboard Overview

### Executive HR Dashboard

This dashboard provides an overview of the workforce through:

* Total Employees
* Attrition Rate
* Average Salary
* Average Satisfaction Score
* Employee Distribution by Department
* Employee Distribution by Job Role
* Gender Distribution
* Age Group Distribution

---

### Attrition Analysis Dashboard

This dashboard focuses on employee turnover by analyzing:

* Attrition by Department
* Attrition by Job Role
* Overtime vs Attrition
* Job Satisfaction vs Attrition
* Average Years at Company
* Attrition KPIs

---

## Key Insights

* Employees working overtime show higher attrition rates.
* Lower job satisfaction is associated with increased employee turnover.
* Salary varies significantly across departments and job roles.
* Workforce distribution differs across departments and experience levels.
* Employee engagement and career development may help improve retention.
