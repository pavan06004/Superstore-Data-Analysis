# Superstore SQL Analysis Project

## Project Overview
This project focuses on analyzing the Superstore sales dataset using SQL to extract business insights related to sales, profit, customers, products, and regions.

The objective is to perform data analysis using SQL queries and understand business performance through different analytical techniques.

## Dataset
Dataset: Superstore Dataset

Records: 9994 rows

Columns include:
- Order Information
- Customer Information
- Product Details
- Sales Data
- Profit Data
- Regional Information


## Tools Used
- MySQL
- MySQL Workbench
- SQL
