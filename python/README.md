# Python Analysis

This folder contains the complete Python workflow for the HR Analytics project. Python was used for data loading, cleaning, exploratory data analysis (EDA), feature engineering, statistical analysis, data visualization, and generating business insights from employee data.

## Workflow

### 1. Data Loading
- Imported the HR dataset using Pandas.
- Explored the dataset structure and data types.

### 2. Data Cleaning
- Checked for missing values and duplicate records.
- Removed unnecessary columns.
- Prepared a clean dataset for analysis.

### 3. Exploratory Data Analysis (EDA)
- Analyzed employee demographics.
- Examined attrition trends.
- Explored salary distributions.
- Studied department and job role distributions.

### 4. Data Visualization
Created visualizations using Matplotlib and Seaborn to analyze:
- Employee attrition
- Salary distribution
- Department-wise employee count
- Overtime impact
- Job satisfaction
- Work-life balance

### 5. Business Insights
Generated actionable insights from the analysis, including:
- Attrition patterns across departments and job roles.
- Impact of overtime on employee retention.
- Relationship between job satisfaction and attrition.
- Workforce distribution by age group and experience level.
- Salary trends and employee retention patterns.

### 6. Feature Engineering
Created additional features such as:
- Age Group
- Salary Band
- Experience Level
- Overall Satisfaction Score

### 7. Statistical Analysis
Performed:
- Descriptive statistics
- Correlation analysis
- Group-wise comparisons
- Attrition rate analysis

## Python Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn

## Output

The processed dataset and analytical insights were used to develop interactive Power BI dashboards for workforce analysis and employee attrition monitoring.
